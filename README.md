\# CleanPage v40.1.1



A comprehensive, privacy-focused browser enhancement script designed to provide a cleaner, safer, and faster browsing experience.



\## Features

&#x20;\* \*\*Intelligent Blocking:\*\* Automatically identifies and removes ads, trackers, and malicious overlays.

&#x20;\* \*\*Privacy Modules:\*\* Includes anti-fingerprinting, WebRTC leak protection, and storage isolation to protect your digital identity.

&#x20;\* \*\*Stealthy Execution:\*\* Utilizes advanced techniques to remain undetected by anti-adblock scripts.

&#x20;\* \*\*Customization:\*\* Full control over allow-lists, block-lists, and external hosts files.

&#x20;\* \*\*User Interface:\*\* A built-in, fully interactive dashboard (SafeView) for real-time statistics and configuration management.



\## Installation

&#x20;1. Install a userscript manager like \*\*TamperMonkey\*\* or \*\*ViolentMonkey\*\* in your browser.

&#x20;2. Click the raw install link for the CleanPage.js file.

&#x20;3. The script will automatically activate on all visited pages.



\## Usage

&#x20;\* Open the dashboard to view blocked stats or modify settings.

&#x20;\* Use the \*\*Zapper Mode\*\* (element picker) to manually remove persistent site elements.

&#x20;\* Import/Export settings via the \*\*Data\*\* tab to sync your preferences across devices.



\## Technical Highlights

&#x20;\* \*\*Mutation Observer:\*\* Dynamically detects and sanitizes new DOM nodes.

&#x20;\* \*\*Fetch/XHR Interception:\*\* Blocks network requests at the API level.

&#x20;\* \*\*Scriptlet Injection:\*\* Provides on-the-fly modification of JavaScript to neutralize tracking scripts.
