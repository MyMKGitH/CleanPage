CleanPage

Overview

CleanPage is a lightweight Tampermonkey userscript developed to improve webpage readability and browsing efficiency.

The script automatically removes distracting page elements and simplifies layouts, allowing users to focus on the primary content.

Features

- Improves webpage readability
- Reduces visual clutter
- Simplifies page layouts
- Lightweight browser-side execution
- Easy installation through Tampermonkey

Technologies

- JavaScript
- Tampermonkey
- DOM Manipulation

Installation

1. Install the Tampermonkey browser extension.
2. Create a new userscript.
3. Copy and paste the contents of "cleanpage.user.js".
4. Save the script.
5. Enable the script and refresh the target webpage.
